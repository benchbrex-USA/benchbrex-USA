## What We Build

BenchBrex is a **compliance-first developer toolchain** — helping engineering teams stay ahead of regulatory requirements, not scramble to meet them at audit time.

| Product | What It Does | Lang |
| :--- | :--- | :---: |
| [**PHANTOM**](https://github.com/benchbrex-USA/PHANTOM) | Core scanning engine — built for speed and reliability | Rust |
| [**BEGUARDIT**](https://github.com/benchbrex-USA/BEGUARDIT) | Code-level compliance guardrails with a CLI-first developer experience | Python |
| [**TokenLens**](https://github.com/benchbrex-USA/TokenLens) | Token and credential exposure detection | TypeScript |
| [**Safepush**](https://github.com/benchbrex-USA/Safepush) | Pre-push hooks that block non-compliant code before it lands | Python |
| [**Meridian**](https://github.com/benchbrex-USA/Meridian) | Network safety and internet policy guardrails | Python |
| [**Octopus**](https://github.com/benchbrex-USA/Octopus) | Web dashboard — onboarding, risk snapshots, and reporting | Python |

BenchBrex is built for the developer, not the auditor.
