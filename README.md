## What We Build

BenchBrex is a compliance-first developer toolchain. We help engineering teams stay ahead of regulatory requirements — not scramble to meet them at audit time.

| Product | What It Does |
|---|---|
| **[PHANTOM](https://github.com/benchbrex-USA/PHANTOM)** | Core scanning engine — written in Rust for speed and reliability |
| **[BEGUARDIT](https://github.com/benchbrex-USA/BEGUARDIT)** | Code-level compliance guardrails with CLI-first developer experience |
| **[TokenLens](https://github.com/benchbrex-USA/TokenLens)** | Token and credential exposure detection |
| **[Safepush](https://github.com/benchbrex-USA/Safepush)** | Pre-push hooks that block non-compliant code before it lands |
| **[Meridian](https://github.com/benchbrex-USA/Meridian)** | Network safety and internet policy guardrails |
| **[Octopus](https://github.com/benchbrex-USA/Octopus)** | Web dashboard — onboarding, risk snapshots, and reporting |

## Philosophy

> *Compliance shouldn't be a department. It should be a property of the code itself.*

We believe the best time to catch a violation is before it's committed — not before a board presentation. BenchBrex is built for the developer, not the auditor.
